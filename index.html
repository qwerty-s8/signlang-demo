How it works

This page captures 160×160 JPEG snapshots and posts them to a FastAPI + ONNX Runtime server, which returns a label and confidence. For capable devices, a pure browser ONNX mode is also available by enabling the script tag and opening with ?mode=local.

  • API inference by default to avoid memory crashes.
  • Local ONNX demo optional via URL switch.
  • No frames are stored on the server.
